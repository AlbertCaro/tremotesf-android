Клиент удаленного управления для BitTorrent-клиента Transmission.

Возможности:
 - Список торрентов
 - Сортировка торрентов
 - Фильтрация торрентов по имени, статусу и трекерам
 - Запуск/остановка/проверка/удаления торрентов, с мультивыбором
 - Добавления торрентов из файлов и magnet-ссылок
 - Возможность выбрать, какие файлы скачивать при добавлении торрента
 - Управления файлами торрентов
 - Добавление и удаления трекеров
 - Список пиров
 - Настройка ограничений торрентов
 - Возможность просматривать и устанавливать настройки удаленного сервера
 - Статистика сервера
 - Управление несколькими серверами
 - Поддержка HTTPS
 - Подключения к серверам с самоподписанными сертификатами (для этого нужно установить сертификат в настройках сервера)
 - Аутентификация с помощью клиентского сертификата

Исходный код: https://github.com/equeim/tremotesf-android
Перевод: https://www.transifex.com/equeim/tremotesf-android
